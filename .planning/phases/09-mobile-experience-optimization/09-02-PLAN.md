2. Scrollable container div with these Tailwind classes:
   - `flex gap-3 overflow-x-auto snap-x snap-mandatory` for horizontal scroll with snap

   - `px-4` for container padding (matches mobile-container pattern)
   - `pb-2` for slight bottom padding

   - `snap-start` for snap alignment
   - Pass through `itemClassName` for width override
