```typescript
export function getSlotConfig(placement: string): SlotConfig | null {
  // placement format: "homepage.sidebar-top" maps to AD_SLOTS.HOMEPAGE.SIDEBAR_TOP
  const [page, position] = placement.split('.');
  const pageKey = page.toUpperCase() as keyof typeof AD_SLOTS;
  const posKey = position.toUpperCase().replace(/-/g, '_');
  const pageSlots = AD_SLOTS[pageKey];
  if (!pageSlots) return null;
  return (pageSlots as Record<string, SlotConfig>)[posKey] || null;
}
```

**src/lib/content-validation.ts — Fix shouldShowAds():**

Replace the always-true stub with real validation (per D-12, D-13, D-14):

```typescript
export function shouldShowAds(content?: string, pageType?: string): boolean {
  // No content provided — check page type only
  if (pageType) {
    if (!validatePageForAds(pageType, !!content)) return false;
  }

  // If content is provided, validate quality
  if (content !== undefined) {
    if (!content || content.trim().length < 200) return false;
    const validation = validateContentQuality(content);
    return validation.isValid;
  }

  // No content, no page type — default true (homepage, etc.)
  return true;
}
```

This makes shouldShowAds():
- Return false for content shorter than 200 chars (per D-13)
- Return false for admin/login/debug/404/error pages via validatePageForAds (per D-14)
- Return false for placeholder content (via validateContentQuality patterns)
- Return true for pages without content requirements (homepage)
