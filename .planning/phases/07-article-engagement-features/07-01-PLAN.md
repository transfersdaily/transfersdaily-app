**ShareButtons.tsx** -- 'use client' component:

Props: `{ url: string; title: string }`.

Render a horizontal row of 3 icon buttons (no text labels, icons only per user spec "minimal"):
1. **Copy Link** -- Lucide `Link` icon. On click: `navigator.clipboard.writeText(url)`, then show "Copied!" text replacing the icon for 2 seconds via useState, then revert.
2. **X/Twitter** -- Lucide `Twitter` icon (or custom X SVG if Twitter icon unavailable; check lucide-react exports). Opens `https://twitter.com/intent/tweet?url=${encodeURIComponent(url)}&text=${encodeURIComponent(title)}` in `window.open(url, '_blank', 'noopener')`.
3. **WhatsApp** -- Lucide `MessageCircle` icon. Opens `https://wa.me/?text=${encodeURIComponent(title + ' ' + url)}` in new tab.

Styling per D-05: Each button is a `<button>` with classes `inline-flex items-center justify-center w-11 h-11 rounded-full text-muted-foreground hover:text-primary hover:bg-muted transition-colors`. The row container uses `flex items-center gap-2 mt-3`.

**ReadingProgressBar.tsx** -- 'use client' component:

Props: `{ targetSelector?: string }` defaulting to `"article"`.

On mount, attach a scroll event listener (throttled to ~60fps via requestAnimationFrame). Calculate progress as: find the DOM element matching targetSelector, compute `(scrollY - elementTop) / elementHeight`, clamp 0-1. Store in useState.

Render: `<div className="fixed top-0 left-0 h-[3px] bg-primary z-50 transition-transform duration-150 ease-out" style={{ width: \`${progress * 100}%\` }} />`. When progress <= 0 or >= 1, set `opacity-0`.

Respect prefers-reduced-motion: Use `window.matchMedia('(prefers-reduced-motion: reduce)')` -- if matches, return null (render nothing) per D-19.

Clean up event listener on unmount.

**index.ts** -- Add exports:
```
export { ShareButtons } from './ShareButtons'
export { ReadingProgressBar } from './ReadingProgressBar'
```
